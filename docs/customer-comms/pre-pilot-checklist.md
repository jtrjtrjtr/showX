# ShowX 0.1 — Pre-Pilot Checklist

> **Who fills this out:** XLAB coordinator (Jindřich or Margaret) with the venue contact
> **When:** Completed at least 5 days before install day
> **Format:** Print this page or share as PDF; tick off items in a call with the SM/tech

---

## Network

- [ ] Gigabit LAN available at FOH position (wired)
- [ ] Spare ethernet port available for FOH Mac (or switch port free)
- [ ] WiFi 5/6 AP within 20m of FOH and operator stations (5 GHz band)
- [ ] WiFi password shared with XLAB coordinator
- [ ] Venue IT confirms mDNS traffic is allowed (Bonjour / `.local` hostnames)
- [ ] Venue IT confirms UDP 5300 and UDP 5305 are not blocked within the LAN

## Hardware

- [ ] FOH Mac: macOS 13 Ventura or newer; 8 GB RAM minimum (M1 or newer recommended)
- [ ] SM device confirmed: iPad 11" or 12.9" (12.9" strongly preferred); Safari or Chrome up to date
- [ ] Operator devices confirmed: iPad or laptop per dept (LX, SX, VIDEO as applicable); modern browser
- [ ] LX console model + firmware version recorded: ______________________
- [ ] SX console model + firmware version recorded: ______________________

- [ ] Draft `.showx` file created by coordinator and sent to SM for review
