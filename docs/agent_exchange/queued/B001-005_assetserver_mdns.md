## Context

ShowX is LAN-first. The FOH Mac hosts:
1. The PWA bundle (served to operator stations as a web app).
2. `_showx._tcp.local` mDNS advertisement so stations find the FOH machine without manual IP entry.

**AssetServer** is the HTTP layer for #1 (plus a couple of system health endpoints + a future hook for modules to add their own static routes and API endpoints). **MdnsService** is the discovery layer for #2. Both are infrastructure consumed by SyncBroker (B001-006 reuses the HTTP server for the Yjs WebSocket upgrade), PairingStore (B001-009 adds an HTTP endpoint), and every PWA station that boots.

Read `docs/specs/protocol_dictionary.md` §8 (mDNS TXT records) and `docs/specs/pairing_auth.md` for the fingerprint semantics. The fingerprint is the SHA-256 of the FOH machine's local secret — generated by SecretStore (B001-004) and exposed via the PairingStore (B001-009). For B001-005, AssetServer + MdnsService take the fingerprint as a constructor arg; B001-011 wires it up from PairingStore at shell boot.

### `src/main/src/shared/version.ts`

Reads the shell's version + build metadata once at boot. Cached.

```ts
import { readFileSync } from 'node:fs';
import { join, dirname } from 'node:path';
import { fileURLToPath } from 'node:url';

export interface ShellVersionInfo {
  version: string;          // from src/main/package.json
  build: string;            // from env SHOWX_BUILD or 'dev'
  electron: string;         // process.versions.electron || 'node'
  node: string;             // process.versions.node
  startedAt: number;        // boot timestamp
}

let cached: ShellVersionInfo | null = null;

export function shellVersion(): ShellVersionInfo {
  if (cached) return cached;
  // resolve src/main/package.json from this file's dir
  const here = dirname(fileURLToPath(import.meta.url));
  let pkg = { version: '0.0.0' };
  try {
    pkg = JSON.parse(readFileSync(join(here, '../../package.json'), 'utf8'));
  } catch { /* fallback to default */ }
  cached = {
    version: pkg.version ?? '0.0.0',
    build: process.env.SHOWX_BUILD ?? 'dev',
    electron: (process.versions as Record<string, string | undefined>).electron ?? 'none',
    node: process.versions.node,
    startedAt: Date.now(),
  };
  return cached;
}
```

### `src/main/src/shared/AssetServer.ts`

```ts
import express, { type Express, type Request, type Response, type Router } from 'express';
import cors from 'cors';
import { createServer, type Server } from 'node:http';
import { createProxyMiddleware } from 'http-proxy-middleware';
import { randomUUID } from 'node:crypto';
import type {
  AssetServer as AssetServerIface, Subscription,
} from 'showx-shared';
import { shellVersion } from './version.js';
import type { Logger } from './Logger.js';

export type AssetServerMode =
  | { kind: 'prod'; pwaDir: string }
  | { kind: 'dev'; viteUrl: string };

export interface AssetServerOptions {
  port?: number;                  // default 5300
  host?: string;                  // default '0.0.0.0'
  mode: AssetServerMode;
  log?: Logger;
  corsOrigin?: (origin: string | undefined) => boolean;
}

interface StaticMount { id: string; slug: string; mountPath: string; router: Router; }
interface ApiMount    { id: string; method: string; path: string; }

export class AssetServer implements AssetServerIface {
  private app: Express;
  private server: Server | null = null;
  private staticMounts: StaticMount[] = [];
  private apiMounts: ApiMount[] = [];
  private apiRouter: Router;
  private moduleRouter: Router;

  constructor(private readonly opts: AssetServerOptions) {
    this.app = express();
    this.apiRouter = express.Router();
    this.moduleRouter = express.Router();
    this.installMiddleware();
    this.installSystemRoutes();
    this.app.use('/api', this.apiRouter);
    this.app.use('/modules', this.moduleRouter);
    this.installPwaServing();
  }

  async start(): Promise<void> {
    if (this.server) return;
    const port = this.opts.port ?? 5300;
    const host = this.opts.host ?? '0.0.0.0';
    this.server = createServer(this.app);
    await new Promise<void>((resolve, reject) => {
      this.server!.once('error', reject);
      this.server!.listen(port, host, () => resolve());
    });
    this.opts.log?.info('AssetServer listening', { port: this.port(), host });
  }

  async stop(): Promise<void> {
    if (!this.server) return;
    await new Promise<void>((resolve) => this.server!.close(() => resolve()));
    this.server = null;
  }

  port(): number {
    if (!this.server) return -1;
    const addr = this.server.address();
    return addr && typeof addr === 'object' ? addr.port : -1;
  }

  baseUrl(): string {
    const host = this.opts.host ?? '0.0.0.0';
    return `http://${host === '0.0.0.0' ? 'localhost' : host}:${this.port()}`;
  }

  httpServer(): Server {
    if (!this.server) throw new Error('AssetServer not started');
    return this.server;
  }

  registerStaticRoute(slug: string, dir: string): Subscription {
    const id = randomUUID();
    const router = express.Router();
    router.use(express.static(dir, { fallthrough: true, index: false }));
    const mountPath = `/${slug}`;
    this.moduleRouter.use(mountPath, router);
    this.staticMounts.push({ id, slug, mountPath, router });
    return { id, unsubscribe: () => this.unmountStatic(id) };
  }

  registerApiRoute(
    method: 'GET' | 'POST' | 'PUT' | 'DELETE',
    path: string,
    handler: (req: Request, res: Response) => Promise<unknown> | unknown,
  ): Subscription {
    const id = randomUUID();
    const wrapped = async (req: Request, res: Response) => {
      try {
        const out = await handler(req, res);
        if (!res.headersSent && out !== undefined) res.json(out);
      } catch (err) {
        this.opts.log?.error('api route threw', { method, path, error: String(err) });
        if (!res.headersSent) res.status(500).json({ error: 'internal_error' });
      }
    };
    (this.apiRouter as any)[method.toLowerCase()](path, wrapped);
    this.apiMounts.push({ id, method, path });
    return { id, unsubscribe: () => this.removeApi(id) };
  }

  private installMiddleware() {
    this.app.use(express.json({ limit: '1mb' }));
    this.app.use(cors({
      origin: this.opts.corsOrigin ?? defaultCorsOrigin,
      credentials: true,
    }));
  }

  private installSystemRoutes() {
    const v = shellVersion();
    this.app.get('/system/health', (_req, res) => {
      res.json({ status: 'ok', version: v.version, uptimeMs: Date.now() - v.startedAt });
    });
    this.app.get('/system/version', (_req, res) => {
      res.json({ version: v.version, build: v.build, electron: v.electron, node: v.node });
    });
  }

  private installPwaServing() {
    if (this.opts.mode.kind === 'prod') {
      this.app.use(express.static(this.opts.mode.pwaDir, { index: 'index.html', fallthrough: true }));
      // SPA fallback for client-side routing
      this.app.get('*', (req, res, next) => {
        if (req.path.startsWith('/api') || req.path.startsWith('/modules') || req.path.startsWith('/system') || req.path.startsWith('/yjs') || req.path.startsWith('/events')) {
          return next();
        }
        res.sendFile('index.html', { root: (this.opts.mode as { pwaDir: string }).pwaDir });
      });
    } else {
      const viteUrl = this.opts.mode.viteUrl;
      this.app.use((req, res, next) => {
        if (req.path.startsWith('/api') || req.path.startsWith('/modules') || req.path.startsWith('/system') || req.path.startsWith('/yjs') || req.path.startsWith('/events')) {
          return next();
        }
        return createProxyMiddleware({ target: viteUrl, changeOrigin: true, ws: true, logLevel: 'silent' })(req, res, next);
      });
    }
  }

  private unmountStatic(id: string) {
    const mount = this.staticMounts.find((m) => m.id === id);
    if (!mount) return;
    this.staticMounts = this.staticMounts.filter((m) => m.id !== id);
    // express does not support unmount; rebuild moduleRouter
    this.rebuildModuleRouter();
  }

  private removeApi(id: string) {
    this.apiMounts = this.apiMounts.filter((m) => m.id !== id);
    // express does not support runtime route removal cleanly; the wrapped handler can short-circuit by checking the list
    // For MVP: track removed routes and short-circuit at runtime
  }

  private rebuildModuleRouter() {
    // Implementation note: replace this.moduleRouter in this.app's stack with a fresh router built from staticMounts.
    // Forge: use express.Router() + reassign + mutate this.app._router.stack to drop the old mount.
    // Acceptable alternative: store a wrapper closure that checks staticMounts; if not present, 404.
    // Pick whichever is simpler; document choice in code comment.
  }
}

function defaultCorsOrigin(origin: string | undefined): boolean {
  if (!origin) return true;     // same-origin / non-browser
  try {
    const u = new URL(origin);
    const host = u.hostname;
    if (host === 'localhost' || host === '127.0.0.1' || host === '::1') return true;
    if (/^10\./.test(host)) return true;
    if (/^192\.168\./.test(host)) return true;
    if (/^172\.(1[6-9]|2\d|3[01])\./.test(host)) return true;
    return false;
  } catch {
    return false;
  }
}
```

### `src/main/src/shared/MdnsService.ts`

```ts
import { Bonjour, type Service, type BrowserConfig } from 'bonjour-service';
import { randomUUID } from 'node:crypto';
import type { MdnsService as MdnsIface, Subscription, MdnsPeer } from 'showx-shared';
import type { Logger } from './Logger.js';

export interface MdnsServiceOptions {
  log?: Logger;
  factory?: () => BonjourLike;     // injectable for tests
}

export interface BonjourLike {
  publish(opts: { name: string; type: string; port: number; txt?: Record<string, string> }): Service;
  find(opts: BrowserConfig, listener?: (svc: Service) => void): { stop(): void; on(ev: string, fn: (svc: Service) => void): void };
  destroy(cb?: () => void): void;
}

interface Advertisement { id: string; service: Service; }
interface Browse        { id: string; stop(): void; }

const SHOWX_SERVICE_TYPE = 'showx';   // becomes _showx._tcp.local

export class MdnsService implements MdnsIface {
  private bonjour: BonjourLike;
  private ads: Advertisement[] = [];
  private browses: Browse[] = [];

  constructor(private readonly opts: MdnsServiceOptions = {}) {
    this.bonjour = (opts.factory ?? defaultFactory)();
  }

  advertise(name: string, port: number, txt: Record<string, string>): Subscription {
    const id = randomUUID();
    const service = this.bonjour.publish({ name, type: SHOWX_SERVICE_TYPE, port, txt });
    this.ads.push({ id, service });
    this.opts.log?.info('mdns advertise', { name, port, txt });
    return { id, unsubscribe: () => this.unpublish(id) };
  }

  browse(serviceType: string, handler: (peer: MdnsPeer) => void): Subscription {
    const id = randomUUID();
    const finder = this.bonjour.find({ type: serviceType }, (svc) => {
      handler({
        name: svc.name,
        host: svc.host,
        port: svc.port,
        txt: (svc.txt as Record<string, string>) ?? {},
      });
    });
    this.browses.push({ id, stop: () => finder.stop() });
    return { id, unsubscribe: () => this.unbrowse(id) };
  }

  async stop(): Promise<void> {
    for (const b of this.browses) b.stop();
    this.browses = [];
    for (const ad of this.ads) ad.service.stop?.();
    this.ads = [];
    await new Promise<void>((resolve) => this.bonjour.destroy(() => resolve()));
  }

  private unpublish(id: string) {
    const ad = this.ads.find((a) => a.id === id);
    if (!ad) return;
    ad.service.stop?.();
    this.ads = this.ads.filter((a) => a.id !== id);
  }

  private unbrowse(id: string) {
    const b = this.browses.find((x) => x.id === id);
    if (!b) return;
    b.stop();
    this.browses = this.browses.filter((x) => x.id !== id);
  }
}

function defaultFactory(): BonjourLike {
  return new Bonjour() as unknown as BonjourLike;
}
```

### TXT record content (per `docs/specs/protocol_dictionary.md` §8)

The caller (B001-011 shell boot) builds the TXT record and passes to `advertise()`. Required keys:
- `role`: `'foh-server'`
- `tier`: `'free' | 'pro'`
- `version`: shell semver
- `hostname`: `os.hostname()`
- `fingerprint`: SHA-256 hex of local secret (provided by PairingStore in B001-009)

Forge should NOT hardcode these in MdnsService — the service is generic. Document the expected TXT keys in a JSDoc on `advertise()` referencing the spec.

## Out of scope

- TLS / HTTPS — ShowX is LAN-trusted; auth happens at Yjs/API layer (B001-006/B001-009).
- WebSocket upgrade routing for `/yjs/*` and `/events/*` — owned by B001-006 SyncBroker (it attaches a `ws` server to AssetServer's `httpServer()`).
- mDNS conflict resolution (same name on network) — bonjour-service handles internally; we don't add UX yet.
- IPv6 mDNS — bonjour-service does both by default; no extra config in MVP.
- HTTP request authentication on `/api/*` — modules attach their own auth middleware if needed; PairingStore-authenticated routes live in B001-009.
- Asset hashing / caching headers — express.static defaults are fine for MVP.
- Real PWA dist — `pwa/dist` doesn't exist yet (B001-012 creates a placeholder). Tests use a tmpdir or skip prod-mode `index.html` serving.

## Notes for Critic

- AssetServer should expose `httpServer(): Server` so SyncBroker (B001-006) can do `wsServer.handleUpgrade()` on it. Verify that getter exists and returns the live `http.Server`, not the Express app.
- Port 0 in tests: confirm `port()` returns the OS-assigned port after start.
- CORS allowlist: write a unit test for `defaultCorsOrigin` covering edge cases (no origin, ::1, 10.x, 172.16.x, 172.31.x, 172.32.x rejected, 192.168.x).
- Static route 404 fallthrough: when a registered static dir doesn't contain the requested file, the request should fall through to next middleware (so SPA fallback can catch /), not return a hard 404 from the static handler.
- `stop()` is idempotent — calling twice doesn't throw.
- mDNS `advertise()` doesn't validate TXT keys; we rely on caller. Critic: make sure no validation rejects the spec'd keys (`fingerprint` etc. — keytar-derived hex strings can be long; bonjour-service should handle but verify).
- The mock `BonjourLike` factory pattern is the Critic's gate for asserting Forge didn't add a hard dependency on real network mDNS in tests.
- Watch for unclosed servers in tests — every test should `await server.stop()` in `afterEach` to avoid port-leak between tests.
